# Simpler Filters
**Version:** 1.0.0

Plugin that replaces the search bar with a dropdown filter system (Tags, Users, Dates, Priorities). It also includes a feature to easily select which columns to display.

## Author
- laimeilleurzamis
- License MIT

## Features
This plugin enhances the board interface by providing dedicated dropdowns for:
- **Location:** Filter tasks by project tags.
- **Priority:** Filter by custom priority levels (e.g., Solved, Weak, Medium, High).
- **Assignee:** Quickly find tasks assigned to specific users or unassigned tasks.
- **Timeframe:** Filter by creation dates such as today, yesterday, this week, or this month.
- **Columns:** Select and validate specific columns to display directly from the filter menu.

## Requirements
- Kanboard >= 1.0.35

## Installation
You have the choice between 3 methods:

1. Install the plugin from the Kanboard plugin manager in one click.
2. Download the zip file and decompress everything under the directory `plugins/SimplerFilters`.
3. Clone this repository into the folder `plugins/SimplerFilters`.

> **Note:** The plugin folder is case-sensitive.
